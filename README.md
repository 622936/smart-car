# 室外智能车导航系统

全国大学生智能汽车竞赛（室外赛道）· **全国二等奖**

基于 ROS 2 + 激光雷达的室外自主导航系统，实现多圈赛道行驶、分段速度控制、红绿灯停车与自动建图。

---

## 项目结构

```
智能车/
├── node_A.py          # 激光雷达 PID 避障（里程计触发版）
├── pid.py             # 激光雷达 PID 避障（坐标触发版，比赛用）
├── go.py              # Nav2 多航点导航客户端（CSV 读取路径点）
├── diantiao.py        # 电调控制节点
├── tt.py              # 工具/测试脚本
├── 12_25改1.py        # 12月25日调参版本
├── slam_gmapping/     # gmapping 建图 ROS 2 包
└── 鲁不鲁队/          # 团队协作版本（含地图文件）
    ├── node_A~D.py    # 分工节点
    ├── gmapping.sh    # 一键启动建图脚本
    ├── map_save.py    # 自动保存地图节点
    └── python_myself/ # 个人调试版本（含历史地图）
```

## 核心算法

### 激光雷达边缘检测 + PID 转向

- 扫描左侧（0°~63°）和右侧（189°~252°）边缘点
- 计算左右边缘中点偏差，输入 PID 控制器
- 输出归一化角速度 `[-1, 1]` 控制转向

```
PID 参数（比赛版）：P=0.38  D=0.10  I=0.0
转向角限幅：±45°
```

### 分段速度控制

赛道分为 7 个阶段，通过**坐标触发**（`pid.py`）或**里程计触发**（`node_A.py`）切换：

| 阶段 | 模式 | 线速度 |
|------|------|--------|
| Seg0 起步直道 | PID | 0.50 m/s |
| Seg1 第一弯 | 固定角度左转 | 0.50 m/s |
| Seg2 S弯 | PID | 0.45 m/s |
| Seg3 直角弯 | 固定角度左转 | 0.57 m/s |
| Seg4 直走段 | PID | 0.30 m/s |
| Seg5 1/4圆 | 固定角度左转 | 0.35 m/s |
| Seg6 冲刺直道 | PID | 0.55 m/s |

### 特殊逻辑

- **红绿灯停车**：到达指定坐标后急停 3 秒，非阻塞定时器实现
- **多圈支持**：第一圈结束自动保存地图，第二圈降速并导航至终点停车
- **自动建图**：`map_save.py` 订阅 `/save_map` topic，触发后自动保存 `.pgm` + `.yaml`

## 依赖环境

- ROS 2 (Humble / Foxy)
- Nav2
- `sensor_msgs`, `nav_msgs`, `geometry_msgs`
- Python 3.10+

---

## 快速启动

```bash
# 启动建图
bash 鲁不鲁队/gmapping.sh

# 启动导航控制（坐标触发版）
ros2 run python_myself node_A

# 启动 Nav2 多航点导航
ros2 run python_myself go
```
